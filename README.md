# treasury-cli-c

C port of [treasury-cli](https://github.com/stevenwilkin/treasury-cli)

## Dependencies

* [dotenv-c](https://github.com/Isty001/dotenv-c)
* [jansson](https://github.com/akheron/jansson)
* [libwebsockets](https://github.com/warmcat/libwebsockets)

## Building

	make

## Running

	./treasury-cli
